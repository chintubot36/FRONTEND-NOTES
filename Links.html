HTML Target Attribute Example

1. target="_self" (Default)

Opens the link in the same tab.

Open Google (_self)

2. target="_blank"

Opens the link in a new tab.

Open YouTube (_blank)

3. target="_parent"

Used mainly with frames or iframes.

Open Wikipedia (_parent)

4. target="_top"

Opens the page in the entire browser window.

Open GitHub (_top)

5. Link without target

Default target is _self.

Open OpenAI
